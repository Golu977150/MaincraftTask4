## ✨ Features

### Core Functionality
- ✅ **Add Tasks** - Create new tasks with a simple input form
- ✏️ **Edit Tasks** - Update task names inline using prompts
- 🗑️ **Delete Tasks** - Remove individual tasks with confirmation
- ✔️ **Mark as Completed** - Toggle task completion status with checkboxes
- 💾 **LocalStorage Persistence** - Tasks survive page refreshes and browser restarts

### Advanced Features
- 🔍 **Real-time Search** - Filter tasks instantly by name
- 🎯 **Smart Filtering** - View All, Pending, or Completed tasks
- 📊 **Task Statistics** - Live counter showing total, completed, and pending tasks
- 🧹 **Clear All** - Bulk delete all tasks with confirmation dialog
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
